# Shopping-Application-Tanishk-Gupta
Build a Spring Boot application with RESTful APIs for managing categories and products using JPA, Hibernate, and an RDB. Implement CRUD operations with a one-to-many relationship (one category can have multiple products) and server-side pagination. Configure the project using annotations, not XML.
